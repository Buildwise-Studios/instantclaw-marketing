# Project Context

## Design System
**ALWAYS read and follow:** `DESIGN.md` before creating any carousel slides.

The design system contains critical rules for:
- 4:5 format space usage (aggressively fill the space)
- Typography (Space Grotesk, large sizes)
- Brand colors and gradients
- Button styling (ShimmerButton from app)
- Layout principles (asymmetry, not centered)

## Key Reminders

### For Instagram Carousels:
1. **Format:** 1080×1350px (4:5 aspect ratio)
2. **Fill Space:** Never leave excessive whitespace
3. **Go Big:** Large typography, large icons, minimal padding
4. **Brand:** Mention OpenClaw, use brand colors correctly
5. **Font:** Space Grotesk only

### Before Starting Any Carousel:
- Read DESIGN.md
- Check post1-revamped/ for reference implementations
- Verify you're using the correct button style from the app
